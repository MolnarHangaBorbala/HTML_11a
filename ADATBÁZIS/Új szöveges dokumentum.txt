1. Melyik évben hány darab fekete autót gyártottak?
SELECT autos.GYARTEV, COUNT(autos.RENDSZAM) AS db
FROM autos
WHERE autos.SZIN='fekete'
GROUP BY autos.GYARTEV

2. Melyik évben gyártották a legtöbb kék ladát?
SELECT autos.GYARTEV, COUNT(autos.RENDSZAM) AS db
FROM autos
WHERE autos.SZIN='kék' AND autos.TIPUS='Lada'
GROUP BY autos.GYARTEV
ORDER BY db DESC
LIMIT 1

3. Melyik színből van a legtöbb Opel?
SELECT autos.SZIN, COUNT(autos.RENDSZAM) AS db
FROM autos
WHERE autos.TIPUS='Opel'
GROUP BY autos.SZIN
ORDER BY db DESC
LIMIT 1

4. Mennyi színenként az Opelek átlagos ára?
SELECT autos.SZIN, AVG(autos.ERTEK) as atlag
FROM autos
WHERE autos.TIPUS='Opel'
GROUP BY autos.SZIN

5. Mennyibe kerül típusonként a legolcsobb és a legdrágább fehér auto?
SELECT autos.TIPUS, MAX(autos.ertek) AS legdrágább, MIN(autos.ertek) AS legolcsobb 
FROM autos
WHERE autos.SZIN='fehér'
GROUP BY autos.TIPUS

6. Szinenként mennyibe kerül a legdrágább Suzuki?
SELECT autos.TIPUS, autos.SZIN, MAX(autos.ertek) AS legdrágább
FROM autos
WHERE autos.TIPUS='Suzuki'
GROUP BY autos.SZIN

7. Mennyi az egyes évben gyártott autok összes értéke, ha csak a 20M fölötti számokat szeretnénk látni?
SELECT autos.GYARTEV, AVG(autos.ertek) AS osszertek
FROM autos
GROUP BY autos.GYARTEV
HAVING osszertek > 20000000

8. Melyik színből hány db van, de csak az 20 és 80 közöttieket?
SELECT autos.SZIN, COUNT(autos.RENDSZAM) AS db
FROM autos
GROUP BY autos.SZIN
HAVING 20 < db > 80
