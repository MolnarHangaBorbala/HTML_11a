1. Melyik évben hány darab fekete autót gyártottak?
SELECT autos.GYARTEV, COUNT(autos.RENDSZAM) as db 
FROM autos 
WHERE autos.SZIN='fekete'
GROUP BY autos.GYARTEV

2. Melyik évben gyártották a legtöbb kék Ladát?
SELECT autos.GYARTEV, COUNT(autos.RENDSZAM) as db 
FROM autos 
WHERE autos.SZIN='kék' AND autos.TIPUS='Lada'
GROUP BY autos.GYARTEV
ORDER BY db DESC
LIMIT 1

3. Melyik színből van a legtöbb Opel?
SELECT autos.SZIN, COUNT(autos.RENDSZAM) as db 
FROM autos 
WHERE autos.TIPUS='Opel'
GROUP BY autos.SZIN
ORDER BY db DESC
LIMIT 1

4. Mennyi színenként az Opelek átlagos ára?
SELECT autos.SZIN, AVG(autos.ERTEK) as atlag 
FROM autos 
WHERE autos.TIPUS='Opel'
GROUP BY autos.SZIN

5. Mennyibe kerül típusonként a legolcsóbb és a legdrágább fehér autó?
SELECT autos.TIPUS, MAX(autos.ERTEK) as maximum, MIN(autos.ERTEK) as minimum
FROM autos 
WHERE autos.SZIN='fehér'
GROUP BY autos.TIPUS

6. Színenként mennyibe kerül a legdrágább Suzuki?
SELECT autos.SZIN, MAX(autos.ERTEK) AS legdragabb
FROM autos
WHERE autos.TIPUS = 'Suzuki'
GROUP BY autos.SZIN

7. Mennyi az egyes évben gyártott autók összes értéke, ha csak a 20 millió fölöttieket szeretnénk látni?
SELECT autos.GYARTEV, SUM(autos.ERTEK) as osszertek
FROM autos
GROUP BY autos.GYARTEV
HAVING osszertek > 20000000

8. Melyik színből hány darab van, de csak a 20 és 80 közé eső darabszámok?
SELECT autos.SZIN, COUNT(autos.RENDSZAM) as db 
FROM autos
GROUP BY autos.SZIN
HAVING db BETWEEN 20 AND 80
